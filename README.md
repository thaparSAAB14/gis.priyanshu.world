A highly optimized portfolio showcasing the intersection of full-stack web engineering and Geospatial Information Systems (GIS). 

Built natively on the Next.js App Router, this repository utilizes advanced styling and spatial component design to host interactive cartography analysis, hardware-accelerated animations, and robust mapping dashboards.

---

## ⚡ Setup Architecture
This portfolio is built on a high-availability serverless setup powered by Vercel and GitHub Actions.

1. Clone the repository:
   ```bash
   git clone https://github.com/thaparSAAB14/gis.priyanshu.world.git
   cd gis.priyanshu.world
   ```

2. Install dependencies & run:
   ```bash
   npm install
   npm run dev
   ```
